## Bubble Sort

 先来直观的看一张动态图,如下bub-1所示:<br/>
  ![bub-1](/res/bubble-sort.gif)
#### 步骤
1. 比较毗邻的两个元素(a,b)
2. 如果a>b,交换他们
3. 重复步骤1和步骤2
#### 时间复杂度
最优时间复杂度:O(N)<br/>
最差时间复杂度:O(N<sup>2</sup>)<br/>
平均时间复杂度:O(N<sup>2</sup>)
#### 空间复杂度
 O(1)
#### 优劣
优:<br/>
简单

劣:<br/>
