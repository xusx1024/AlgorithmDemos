#### 步骤
插入排序和安排手里的扑克牌情形差不多.
1. 从手里的某一张牌开始
2. 选取下一张牌,插入准确的排序中
3. 对手中所有的牌重复此步骤
#### 时间复杂度
最优时间复杂度:O(N)<br/>
最差时间复杂度:O(N<sup>2</sup>)<br/>
平均时间复杂度:O(N<sup>2</sup>)
#### 空间复杂度
 O(1)
#### 优劣
优:<br/>
稳定,简单

劣:<br/>
效率低
