#### 步骤
给定数组大小为N,并给定一个序列的元素位置P.
1. 找到小于P的元素放在左边
2. 找到大于P的元素放在右边
3. 重复步骤1,步骤2
#### 时间复杂度
平均时间复杂度:O(NlogN)
最差时间复杂度:O(N<sup>2</sup>)
最好时间复杂度:O(NlogN)

#### 空间复杂度
 O(logN)
#### 优劣
不稳定,比较复杂.
