  ![sel-1](/res/selection-sort-animation.gif)
#### 步骤
给定数组大小为N,起始位置为L
1. 找到N-L+1数组中的最小值
2. 和L交换位置
3. L++,重复步骤1,直到L=N-2
#### 时间复杂度
时间复杂度:O(N<sup>2</sup>)
#### 空间复杂度
 O(1)
#### 优劣
优:<br/>
简单

劣:<br/>
效率低

就是冒泡的一个变体吧.都是需要和剩下的元素一一比较.

## HeapSort

即具有归并排序的O(NlgN),又具有插入排序的空间原址性.

使用完全二叉树实现二叉堆.

二叉堆分为最大堆和最小堆.

一般都用数组来表示堆.



#### 时间复杂度
时间复杂度:O(NlgN)
 
#### 步骤

从数组读入元素,形成一个原始堆
