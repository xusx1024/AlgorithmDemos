## 要求

- 要求常用算法,拿过来就可以写出
- 要求不常用的,拿起书,看10分钟就可以理解算法

#### 应该熟练掌握的常用算法

- 各种排序算法
- 线性表(栈,队列)的插入和删除
- 二叉树的遍历(前序,中序,后序)
- 图的遍历(深度优先,广度优先)
- 二分法查找,排序二叉树,Hash和查找(处理冲突的方法)
